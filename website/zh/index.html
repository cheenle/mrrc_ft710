核心功能

完整的 FT-710 远程控制

从 FT4222 SPI 真实频谱瀑布图到双向 Opus 音频、CAT 控制、移动端优化 PTT —— 每个功能都专为 FT-710 打造。

真实 FT4222 SPI 频谱

通过 FT4222 SPI 从 FT-710 DSP 芯片获取 850 点 FFT 频谱，~21fps。120 行历史瀑布图，黑色→蓝色→绿色→黄色→红色色图。双模式：FT4222 不可用时 S-meter 合成回退。

  • FT4222 SPI 直读 (libft4222 + libftd2xx)
  • 4096 字节数据块，0xFF01EE01 同步标记
  • ~21fps 真实频谱，~30fps S-meter 回退

双向 Opus 音频

RX: FT-710 USB 音频 → PyAudio 采集 (48kHz Int16) → Opus 64kbps 编码 → 浏览器 AudioWorklet。TX: 浏览器麦克风 (48kHz) → Opus 64kbps CBR → 服务端 48→44.1kHz 重采样 → PyAudio → 电台。

  • RX 48kHz 高保真 AudioWorklet 播放
  • TX 64kbps CBR，60ms 抖动缓冲
  • 优雅 PTT 释放 — 话尾不被切断

完整 CAT 控制

通过增强型 COM 端口以 38400 baud 直驱串口 CAT。40+ 指令：频率、15 种模式、23 种滤波器宽度、VFO A/B、分频、前置放大、衰减器、NR/NB/AN、压缩器、天调、AF/RF/Mic 增益、S-meter、多表头。

  • 无需 Hamlib / wfview / 中间件 — 直驱串口
  • 7 层轮询 (~296 字节/秒)
  • 脏字段变更追踪最小化 WebSocket 流量

移动优先 UI

专为 iPhone Safari 优化：safe-area 适配、PWA manifest + Service Worker 离线缓存、Wake Lock 屏幕常亮、全屏切换。触控握持 PTT，松手即断。

  • PWA manifest + Service Worker 离线缓存
  • Wake Lock API (屏幕常亮)
  • Canvas S-meter S1–S9+60dB + 多表头 (PWR/ALC/SWR/Id/Vd)

7 层后台轮询

38400 baud 下 100ms→5s 智能分频，0.25s 超时上限，主动让步 PTT。轮询层级：100ms(频率/模式/S-meter)、500ms(TX 表头零化)、2s(滤波器/增益/NR/NB)、5s(电压/电流/压缩器)。

  • ~296 字节/秒总串口占用
  • TX 切 RX 自动零化 TX 表头
  • 每条命令 0.25s 超时 — 绝不阻塞 PTT

多层 PTT 安全防护

触控握持发射 → 松手即断。以 200ms 间隔进行三重 TX0 验证。客户端断开服务端强制 RX。浏览器卸载信标在标签页关闭时发送 TX0;。

  • 每次 PTT 释放后三重 TX0 验证
  • 最后客户端断开时服务端强制 RX
  • RF 断开前优雅 TX 音频排空

快速开始

5 分钟搭建运行

克隆仓库、安装依赖、启动服务器、打开浏览器 — 只需四步。

克隆仓库

从 GitHub 获取最新代码

git clone https://github.com/cheenle/mrrc_ft710.git

安装依赖

Python 3.12+，包含 FastAPI、Uvicorn、PySerial、PyAudio、NumPy

cd mrrc_ft710 && pip install -r requirements.txt

启动服务器

设置串口并启动 — 自动检测 FT-710 USB 设备

FT710_SERIAL_PORT=/dev/cu.usbserial-XXXX python server.py

打开浏览器

同一网络内的任意设备 — 手机、平板、电脑。默认密码：ft710

https://服务器IP:8888 → 登录 → 全功能控制
在 GitHub 上查看
运行环境

你需要什么

八重洲 FT-710 USB 线 (A→B 或 C→B) Python 3.12+ macOS / Linux / RPi 现代浏览器 FTDI 库 (频谱功能)

使用真实 FT4222 SPI 频谱数据需要安装 libft4222 + libftd2xx（可从 wfview 应用包或 FTDI 官网获取）。CAT 控制和音频功能无需任何 FTDI 库即可正常工作。
