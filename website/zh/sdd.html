V1.6 变更重点

RX 零音频看门狗

服务端检测到位精确全零的 RX 音频（静噪开启下持续 ≥20 秒）即在状态栏亮出"无声"警告并写日志——电台 USB 音频卡死再也无法躲在 CBR 码率指示后面装正常。

鉴权失效自动跳登录

过期 token 的 WebSocket 握手被 HTTP 403 拒绝（浏览器只收到 1006，永远不是 4001）。控制通道现在先查 /api/auth/check，确认失效后跳转登录页，不再拿着死 token 无限重试。

死载波修复

TX 音频属主连接在发射中断开时，服务端现在强制 TX0——关闭了此前只能等控制通道断开的无调制载波窗口。

TX 音频文档归真

文档与运行时对齐：TX Opus 实为 64kbps CBR complexity=5；AD-011 重写——48kHz 编解码域经帧对齐桥接（960↔882）到 FT-710 原生 44.1kHz USB 音频。
