系统架构

  📱 浏览器 (iPhone / Desktop / Tablet)
   │  ↕ HTTPS + WebSocket × 4
  🖥️  MRRC FT-710 Server (Python FastAPI + Uvicorn)
   │
   ├── 📡 Serial CAT (USB Enhanced COM Port, 38400 baud)
   ├── 🔬 FT4222 SPI → scope_pipe 子进程 → 真实 FFT 频谱
   ├── 🎙️  PyAudio USB Audio (RX 录音 / TX 播放)
   └── 🗜️  libopus (ctypes): RxOpusEncoder + TxOpusDecoder
  📻  Yaesu FT-710 HF/50MHz Transceiver

不依赖 wfview / Hamlib / 任何外部电台中间件。CAT 指令直接走串口，频谱数据从 FTDI FT4222 SPI 芯片读取，音频通过电台内置 USB 声卡捕获/播放。

快速开始

# 1. 安装依赖
cd FT710
pip install -r requirements.txt

# 2. 启动服务 (macOS)
FT710_SERIAL_PORT=/dev/cu.usbserial-0121DB3A0 python server.py

# 2. 启动服务 (Linux / Raspberry Pi)
FT710_SERIAL_PORT=/dev/ttyUSB0 python server.py

# 3. 打开浏览器
#    访问 https://<服务器IP>:8888
#    默认密码: ft710

PTT 安全架构

电台发射安全是最高设计要求。MRRC FT-710 实现了多层防护：

  1. 触控握持 → 松手即断：PTT 采用 touch-and-hold 交互，鼠标离开/触摸取消立刻释放。
  2. 服务端 TX 状态轮询：0.5s 间隔查询 TX;，TX→RX 转换时立刻广播并清零 TX 表头。
  3. PTT 看门狗：客户端 500ms 定时器验证 RX 状态，异常时重发 TX0;（最多 3 次）。
  4. 服务端强制 RX：当最后一个客户端断开时如果电台仍在 TX，服务器自动发送 TX0;。
  5. 浏览器卸载信标：beforeunload + pagehide 事件在页面关闭时强制 TX0;。
  6. TX 音频优雅释放：PTT 松开后先写入剩余音频到 DAC 等待播完（50ms），再断开 RF，避免话尾被切。
