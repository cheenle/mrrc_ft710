What Changed In V1.3

Priority CAT Command Path

`send_priority_set_command()` + `_cancel_polls` allow PTT/TUNE commands to preempt poll queries and reduce control latency.

Tune + Tuner Sequence

Tune flow now uses `TX2` with `AC003` (start tuning) and exits with `AC000` + `TX0` for clean tuner state transitions.

Expanded Telemetry Model

Added `MS`/`AO`/`RG0` fields and `RI0` status mapping (Hi-SWR, recorder, RX/TX state, scan, squelch-open).

7-Task Poll Scheduler

Polling now explicitly separates IF, VFO, TX status, TX meters, settings, slow telemetry, and connection watchdog tasks.
