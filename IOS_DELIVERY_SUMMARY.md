## 📱 使用方法

### 快速开始
```bash
cd mrrc_ft710/FT710Mobile
open FT710Mobile.xcodeproj
# 选择模拟器或真机 → Cmd+R
```

### 链接 libopus
1. 下载 libopus iOS 静态库
2. 添加到 Xcode 项目 Linked Frameworks
3. 设置 Header Search Paths

### 运行测试
```bash
xcodebuild test \
  -project FT710Mobile.xcodeproj \
  -scheme FT710Mobile \
  -destination 'platform=iOS Simulator,name=iPhone 15'
```
